
**Best for:** If you have access to college lab

---

### ✅ WAY 3: Install Android Studio Yourself (PROPER - 30 minutes)

```
1. Download Android Studio from: https://developer.android.com/studio
2. Install it (takes 10-15 minutes)
3. Open Android Studio
4. File → Open → WiFiCollector folder
5. Wait for Gradle sync (2-3 minutes, automatic)
6. Build → Build Bundle(s) / APK(s) → Build APK(s)
7. Find APK in: WiFiCollector/app/build/outputs/apk/debug/app-debug.apk
8. Copy to phone and install
9. DONE!
```

**Best for:** If you want to learn and have time

## 📱 HOW TO INSTALL APK ON YOUR PHONE

1. Copy `app-debug.apk` to your phone (via USB or Google Drive)
2. Open the file on your phone
3. If it says "Install blocked" → Settings → Allow from this source
4. Click Install
5. Open the app!

## ❓ TROUBLESHOOTING

**Q: None of my teammates have Android Studio**
A: Go to college computer lab OR use online build service (see EASIEST_WAY.md)

**Q: I want to modify the code**
A: Edit `app/src/main/java/com/project/wificollector/MainActivity.kt`

**Q: The APK won't install on my phone**
A: Settings → Security → Allow installation from unknown sources

**Q: App crashes after install**
A: Settings → Apps → WiFi Collector → Permissions → Allow Location
